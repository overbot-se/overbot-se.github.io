OverBot is customizable, but it's not the easiest.



Custom Prefix

Setting a custom prefix is the easiest key to set. Since GuideBot uses a weird settings handler, the command is weird. To set a custom prefix, run the command ~set edit prefix [new prefix here.] If you want to view the existing prefix, ping the bot.

Mod-Log Channel

In order to change the Mod-Log Channel, run the following command, [prefix]set edit modLogChannel [channel name]
Do not put # in front of the channel name because the bot handles names, not ids.
