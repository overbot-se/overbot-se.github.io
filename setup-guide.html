You can customize OverBot, here's how:

Please note that this guide is prone to change!

Custom Prefix

Setting a custom prefix is the easiest key to set. Since GuideBot uses a weird settings handler, the command is weird. To set a custom prefix, run the command
~set edit prefix [new prefix here.]
If you want to view the existing prefix, ping the bot.


Mod-Log Channel

In order to change the Mod-Log Channel, run the following command,
[prefix]set edit modLogChannel [channel name]
Do not put # in front of the channel name because the bot handles names, not ids.


Welcome Channel

In order to change the Welcome Channel, run the following command,
[prefix]set edit welcomeChannel [channel name]
Again, do not put the # symbol in front of the channel name as OverBot does not handle channel ids.


Welcome Message

In order to change the welcome message, run the following command,
[prefix]set edit welcomeMessage [message]

Enable Welcome Message

If you cannot see the Welcome message, it's probably not enabled, enable it with the following command,
[prefix]set edit welcomeEnabled [true/false]

FAQ

Why aren't the moderation commands working?

If the moderation commands don't work, you probably didn't set a modLogChannel, please refer to the Mod-Log Channel section above to find out how to set the modLogChannel key.

Why isn't it showing a Welcome message?

Regardless if you set a welcome message or not, you still need to have set the welcome channel, please refer to the Welcome Channel section above to find out how to set the welcomeChannel key.
Why isn't the welcome message showing?
This is probably because the welcome message isn't enabled, run [prefix]set edit welcomeEnabled true
